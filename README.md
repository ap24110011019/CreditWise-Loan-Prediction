# CreditWise – Loan Approval Prediction

## Overview
CreditWise is a machine learning project that predicts loan approval status based on applicant details such as income, education, employment status, and credit history.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Loan approval prediction
- Model evaluation and accuracy testing

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains applicant information used to determine loan approval eligibility.

## Future Improvements
- Web deployment using Streamlit
- Interactive frontend UI
- Real-time prediction system
